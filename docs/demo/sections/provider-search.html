Workshop talking points

  • Hybrid search beats keyword. Specialty synonyms ("heart doctor" → cardiology) and language preferences plug in via Azure AI Search vector + semantic ranker.
  • Always disclose the source. The directory file + refresh timestamp is shown alongside each result so members know how fresh the data is.
  • Distance is a feature, not just a sort. Pre-filter to plan service area before re-ranking by relevance.
