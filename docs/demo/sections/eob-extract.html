EOB Document Extraction

Paste an Explanation of Benefits and pull out the structured fields. In production this is a call to Azure AI Document Intelligence; here we use regex heuristics so the demo runs offline.

Paste an EOB

Why this matters

  • Reduces handle time. Members call about EOBs they can't read; the copilot pre-extracts the key fields before the agent picks up.
  • Confidence-scored fields. In production, every extracted field carries a confidence score; below threshold → human review queue.
  • Schema-stable. The extractor returns the same JSON shape regardless of EOB layout, so downstream logic is simple.
