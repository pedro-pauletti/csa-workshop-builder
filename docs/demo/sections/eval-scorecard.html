The harness ships with 50 queries; we show 7 here. Out-of-scope queries are expected to fail groundedness — that's a feature, not a bug.

How it ties back to the workshop

  • Compliance gating. No release reaches production unless groundedness ≥ 0.90 on the gold set.
  • Drift detection. The same set runs nightly; a 5-point drop pages the on-call engineer.
  • Human-in-the-loop. Failed queries are routed to SMEs to label "expected refusal" vs. "real miss".
