# Agenda — Generic AI workshop

> The single source of truth for navigation. One line per section. Order is
> a narrative arc: context → architecture → demos → ops → wrap-up.

- Introduction: Workshop goals, audience, requirements overview.
- Architecture: End-to-end reference architecture for a grounded AI assistant.
- Demo 1 — Conversational AI: Embedded chat with citations and trace.
- Demo 2 — Search: Ranked retrieval over a small mock corpus.
- Demo 3 — Workflow: Step-by-step agent-assisted resolution timeline.
- Demo 4 — Document analysis: Structured extraction from a sample document.
- Demo 5 — Evaluation: Scorecards for groundedness, relevance, harm.
- Observability: Traces, token usage, cost per resolved task.
- Roadmap and next steps: From workshop to pilot.
