# Workshop Agenda — Microsoft AI Foundry (sample)

## Sections

- Introduction and customer context: Frame the AI adoption journey, audience and outcomes for the workshop.
- Foundry building blocks: Explain agents, model catalog, evaluations, tracing and governance.
- Solution architecture: Show a reference architecture for an enterprise grounded agent on Foundry.
- Guided web app experience: Demonstrate the workshop app navigation, presenter mode and section structure.
- Demo 1 — Grounded chat agent: Show a chat agent grounded on a curated knowledge source with citations.
- Demo 2 — Hybrid search: Show retrieval, ranking and source grounding behind the agent.
- Demo 3 — Multi-step agent workflow: Show a workflow with tool use and a step-by-step execution timeline.
- Demo 4 — Document analysis: Show structured extraction and summarization from a sample document.
- Evaluation and quality: Show a reproducible evaluation set with prompts, expected outputs and scorecards.
- Tracing and observability: Show end-to-end traces for a multi-step agent invocation.
- Governance and operations: Show secrets handling, dataset versioning and policy guardrails.
- Wrap-up and next steps: Summarize requirement coverage and propose a 30/60/90 day adoption plan.
