# Workshop Agenda — Microsoft Fabric (sample)

## Sections

- Introduction and customer context: Frame the unified data estate vision and the audience for the workshop.
- Fabric building blocks: Explain OneLake, Lakehouse, Pipelines, Notebooks, Semantic Models, Power BI and Real-Time Intelligence.
- Solution architecture: Show a reference Bronze/Silver/Gold architecture on OneLake with downstream BI and real-time.
- Guided web app experience: Demonstrate the workshop app navigation, presenter mode and section structure.
- Demo 1 — Ingest with Pipelines: Show a pipeline ingesting a sample source into the Bronze layer of a Lakehouse.
- Demo 2 — Transform with Notebooks: Show a PySpark notebook curating Silver and Gold Delta tables.
- Demo 3 — Semantic model and Power BI: Show building a semantic model and a report consuming Gold tables.
- Demo 4 — Real-Time Intelligence: Show ingesting an event stream into an Eventhouse and querying with KQL.
- Code walkthrough: Walk through the key notebook and pipeline snippets.
- Governance and operations: Show workspace roles, sensitivity labels, lineage and OneLake security.
- Evaluation and quality: Show data quality checks and observability across the medallion layers.
- Wrap-up and next steps: Summarize requirement coverage and propose a phased Fabric adoption plan.
