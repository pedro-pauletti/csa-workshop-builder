Azure AI Search · Contoso Outdoor

A working workshop on retrieval for grounded AI

Six interactive sections that walk a CSA and a customer architect through building Azure AI Search as the retrieval layer for a generative-AI application. The example customer is Contoso Outdoor, a fictional e-commerce retailer that wants natural-language product search and a grounded chat assistant on top of its catalog.

1. What is Azure AI Search

The service in one screen — when to use it, what it ships with.

2. Index design & ingestion

Fields, analyzers, vectorizers. Inspect the Contoso schema interactively.

3. Vector & hybrid search

Type a real query; toggle between vector, BM25, and hybrid scoring.

4. Semantic ranker

Side-by-side before/after the L2 semantic ranker, with captions.

5. Grounded chat (RAG)

A chat that cites Contoso catalog items pulled from the index.

6. Evaluation & next steps

Quality metrics, a fixed test set, and a 30/60/90 roadmap.
This is a mock. No real Azure subscription is required. Every response is served from local JSON so the workshop runs offline. The csa-workshop-builder tutorial walks you through generating an app exactly like this one for your own customer.
